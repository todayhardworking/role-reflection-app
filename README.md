# Role Reflection App

A Next.js 14 project using the App Router and Tailwind CSS.

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) to see the app.

## Available Pages

- `/` – Home page showing **Role Reflection App**.
- `/signin` – Basic email/password sign-in form UI (no functionality yet).

## Firebase Setup

Firebase is initialized in `lib/firebase.ts`. Environment variables are intentionally left empty; populate them with your Firebase project values before enabling authentication.
